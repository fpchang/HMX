## 🚀 快速开始

### 安装方式
1. ​**完整导入**：  
   在 [插件市场](https://ext.dcloud.net.cn/plugin?name=lime-ui) 点击「试用」按钮，一键集成基础组件库     
2. ​**按需导入**：  
   访问组件详情页 → 点击「导入」获取组件独立包

> 📦 推荐方案：基础组件库 + 按需加载特定组件

## ⚙️ Vue2 兼容配置
在`main.js`中添加：
```js
// Vue2 项目需要添加
import Vue from 'vue'
import VueCompositionAPI from '@vue/composition-api'

Vue.use(VueCompositionAPI)
```
详细配置参考：[composition-api 文档](https://uniapp.dcloud.net.cn/tutorial/vue-composition-api.html)
